사용자 화면 노출 금지 문구

empty-state는 타이틀 없이 문장형 본문만, compact 좌측정렬 row로 표시. 내부 상태값(11종)→사용자 문구 매핑:

not_in_form → 해당 문서에서는 제공되지 않습니다. (공시 기준 차이)
not_provided → 원문에서 해당 항목을 확인할 수 없습니다.
unconnected → 현재 연결된 데이터가 없습니다.
needs_structuring → 원문 표 정형화가 필요합니다.
not_applicable / connected_zero → 원문 기준 해당 보수 지급 내역이 없습니다.
unverified → 원문 기준 확인이 필요합니다. · check → 원문 구조상 추가 확인이 필요합니다.
policyOnly → 보수 정책만 제공됩니다. · partial → 일부 항목만 연결되었습니다.
